머지의 종류띠
3way : 기본
fast-forward : 머지하려는 가지가 기준가지의 끝부분에서 떨어져 나갔을때 / 브랜치의 커밋들 압축해서 머지
rebase : 머지하려는 가지를 제일 앞대가리에 가져다놓기 ( 1. new에서"git rebase main" 2. main에서 "git merge new")
squash & merge : 브랜치의 커밋들 압축해서 머지 (메인에서 "git merge --squash new)

git branch -M main
git remote add 변수명(origin) 주소
git push -u https://github.com/ohsedu/git_prac.git main
git push -u(기억하라) 저장소주소 브랜치명

push 전에 pull부터 하쇼

git pull : fetch + merge
git fetch : 원격저장소 신규커밋 가져와주세요 
git merge : 내 브랜치에 머지
